Detailed Description

An attribute is another basic data element that defines part of cell tuple in a TAR, A TAR has a set of dimensions. A set of values for each one of the dimensions (a.k.a. indexes) determines a position in the TAR. In every TAR position there can a cell or a tuple of values. The attributes in a TAR define which values exist in its tuples, like a column in the relational model.
